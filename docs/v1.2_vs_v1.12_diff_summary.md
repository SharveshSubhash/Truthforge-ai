# TRUTHFORGE AI v1.2 vs v1.12 Diff Summary

This note captures the meaningful file-level differences between the older `v1.2` download and this `v1.12` repo, excluding `agency-agents/`.

## Current Repo Status

The `v1.2` versions of the files below have now been restored into this `v1.12` repo at their original paths.

Follow-up compatibility fixes were also applied so the restored logic works cleanly with the rest of the repository:

- `agents/explainability_agent.py`
  `observe` and `reason` were made backward-compatible by giving them defaults, so both the restored `v1.2` schema and the newer `v1.12` integration fixtures can coexist.
- `agents/transcript_processing_agent.py`
  the POS-dependent `JUDGE` matcher pattern was replaced with a safer title-case pattern so the blank spaCy fallback pipeline no longer crashes when no full model is installed.

Verification:

- full test suite passed after restoration and compatibility fixes
- result: `220 passed`
